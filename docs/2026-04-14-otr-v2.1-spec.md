| HyWorld Field | Source Token | Extraction |
|---|---|---|

| `audio_anchor_id` | — | Deterministic mapping post-mastering |

## Parser Output Contract

```yaml
scenes:
  - scene_id: string
    persistence_key: string          # hash(world_prompt + time_of_day)
    world_prompt: string             # spatial only
    time_of_day: string
    characters_present: list[string]
    scene_transition_type: enum [cut, crossfade, smash_cut, dissolve]
    audio_anchor_id: string
    beats:
      - speaker: string | null
        line: string | null
        sfx: string | null
        mood: enum [tense, melancholy, action, dialogue, transition]
        duration_sec: float          # exact match to TTS audio
        shot_type: enum [wide, mid, over_shoulder, pov, silhouette, tracking]
        motion: enum [static, slow_push, slow_pull, pan_left, pan_right, handheld_tilt]
        warning: boolean
```

## Shot Constraint (C6/V3)

| Condition | Allowed Shots | Forbidden |
|---|---|---|
| `speaker != null` | `over_shoulder`, `silhouette`, `wide`, `tracking` | `mid`, `pov` |
| `speaker == null` | Any | None |

Validator overrides violations → `wide` + `static` + `warning: true`.

**Sliding window per beat:**
```
[SCENE_MASTER_CONTEXT]  ← original environment.description, always present
[PREVIOUS_BEAT]         ← context only, do not parse
[TARGET_BEAT]           ← extract schema here
[NEXT_BEAT]             ← context only, transition anticipation
AUDIO DURATION: <float>
```

## Failure Containment

1. **Regex sanitize** — strip markdown fences, fix commas/quotes
2. **Pydantic validate** — full schema check + `speaker ∈ characters_present`
3. **Self-correct** — feed error trace back to parser, max 2 retries
4. **Circuit breaker** — inject placeholder, continue pipeline:
```yaml
world_prompt: "Abstract cinematic void, soft cinematic lighting, depth of field."
shot_type: wide
motion: static
scene_transition_type: dissolve
warning: true
```

## Pipeline Flow

```
ScriptWriter → script_json ─┬→ AudioPipeline (UNTOUCHED, C7)
                             └→ StoryParser → StructuredVisualScript
                                  → OTRWorldGen (subprocess, spawn)
                                  → ShotSelector (no-lip-sync enforced)
                                  → OTRWorldRender (subprocess, spawn)
                                  → LTX Supplement (action bursts only)
                                  → AV Mixer → episode.mp4
```

## Build Order

**Phase 0** (pre-reqs):
v2.0-alpha stable · HyWorld 2.0 confirmed · 5-10 golden test episodes annotated

**Phase 1** (start now):
`OTR_StoryParser` node · sliding window + scene master context · Pydantic + 4-tier containment · persistence_key (hash) · continuity_score (cosine sim, not LLM) · golden corpus accuracy: structural ≥99%, duration 100%, mood/world ≥85%

**Phase 2** (blocked on HyWorld 2.0):
`OTRWorldGen` subprocess · world cache · `ShotSelector` with lip-sync grammar · camera path gen

**Phase 3**:
`OTRWorldRender` subprocess · extend `SignalLostVideo` for world frames · LTX narrowed to action · ffmpeg mux

**Contingency Phase B** (only if parser mood/world accuracy <85%):
Add `[SETTING:]` `[MOOD:]` markers to story prompt · audio strips markers pre-TTS · visual reads them directly
